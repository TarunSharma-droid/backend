
NPM --> Node Package Manager

It is a large store that comes with Node.js which contains a large number of packages (libraries).
By using npm commands, we can start a Node.js project as well as install and manage packages (libraries) for Node.js projects.

Example libraries: Express, Mongoose, Lodash, Axios, React etc.

Express and Mongoose are stored in the npm registry (an online database of packages).
When you install them, npm downloads them to your project.

e.g.

npm init                     --> initializes a Node.js application using npm
npm install express          --> to install Express for the project
npm install mongoose         --> to install Mongoose for the project

import express from "express"   --> to use Express
import mongoose from "mongoose" --> to use Mongoose

When we run --> npm init --> a Node.js project is initialized and it asks for basic details
such as name, description, keywords etc. for the project.
